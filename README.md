# 🤖 AI-Toolkit
 
🌟 轻量级人工智能工具集，集成经典算法与前沿模型实践。包含：
 
- 📚 机器学习基础模块（线性回归/决策树等）
- 🌐 NLP预处理工具（分词/词向量可视化）
- 🖼️ 计算机视觉工具（图像增强/目标检测辅助）
- 🚀 深度学习模板（PyTorch/TensorFlow快速搭建）
 
💻 安装：`pip install ai-toolkit`  
📝 使用：查看Jupyter示例 notebook  
✨ 特点：模块化设计+中文注释+可视化调试界面
